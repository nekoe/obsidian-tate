# TATE — 縦書きモード for Obsidian

Obsidian 用の縦書きエディタプラグインです。`writing-mode: vertical-rl` による本格的な縦書き表示と編集に対応しています。

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/T6T71YGS7B)

## 機能

### 青空文庫記法によるルビ・縦中横・傍点

[青空文庫](https://www.aozora.gr.jp/)の注記記法に対応しています。

| 記法 | 表示 |
|------|------|
| `｜東京《とうきょう》` または `東京《とうきょう》`（漢字連続） | ルビ（振り仮名） |
| `２０２５［＃「２０２５」は縦中横］` | 縦中横（数字などを横向きに） |
| `春［＃「春」に傍点］` | 傍点（胡麻点） |

**コマンドパレットから設定する**

テキストを選択してコマンドパレット（`Ctrl+P` / `Cmd+P`）を開き、以下のコマンドを実行します。

- `選択テキストにルビを設定 (ruby)`
- `選択テキストを縦中横にする (tate-chu-yoko: tcy)`
- `選択テキストに傍点を付ける (bouten)`

**インライン編集**

カーソルをルビ・縦中横・傍点の要素に移動すると、Aozora 生テキストがその場で展開されます（Obsidian の Markdown エディタと同様の挙動）。カーソルが要素外に移動すると自動的にレンダリングに戻ります。展開中に **Enter** キーを押してもレンダリングに戻ります。

### 自動字下げと入力補助

縦書きビューでの入力を補助する4つの独立した設定があります。いずれも全角スペース（U+3000）を実際にファイルに書き込みます（CSS による見た目だけのインデントではありません）。

| 設定 | 動作 |
|------|------|
| **半角スペースを全角スペースに変換** | スペースキーで入力した半角スペースを全角スペース `　` に変換します。ペーストは対象外。 |
| **行頭入力時に自動字下げ** | 行頭で文字を入力すると、全角スペースを1文字自動挿入します（通常入力・日本語IMEの両方に対応）。 |
| **字下げを前の段落に揃える** | 改行して新しい段落を作ったとき、直前の段落の行頭スペース数と同じ数の全角スペースを自動挿入します。上の設定とは独立して動作します。 |
| **行頭カッコのインデントを削除** | 行頭のスペースの直後に全角開きカッコ（`「『（` など）を入力すると、行頭のスペースを1文字削除します。日本語組版の慣習に従い、カッコは字下げしない形に整えます。 |

4つすべてデフォルトで ON です。

### 禁則処理

行頭・行末に置けない文字（句読点・括弧など）の扱いをブラウザの `line-break` プロパティで制御します。設定から以下のルールセットを選択できます。

| ルール | 説明 |
|--------|------|
| `Normal` | 一般的な禁則ルールを適用（デフォルト） |
| `Strict` | 最も厳格。小書き仮名（っ・ゅなど）も行頭に置かない |
| `Loose` | 新聞スタイル。制限を緩めて改行を優先 |
| `Anywhere` | 禁則なし。どこでも改行を許可 |

## インストール（開発版）

1. リポジトリをクローンしてビルドします

```bash
git clone https://github.com/nekoe/obsidian-tate
cd obsidian-tate
npm install
npm run build
```

2. ビルド成果物を Obsidian のプラグインフォルダにコピーします

```bash
cp main.js manifest.json styles.css {YOUR_VAULT_PATH}/.obsidian/plugins/tate/
```

3. Obsidian の「設定 → コミュニティプラグイン」でプラグインを有効化します

## 使い方

- コマンドパレット（`Ctrl+P` / `Cmd+P`）から `縦書きビューを開く` を実行して縦書きビューを開きます
- 縦書きビューはタブとして開かれ、アクティブなファイルを自動で読み込みます
- 編集内容は通常のObsidianファイルと双方向に同期されます

## 制限事項

- **傍点に続く IME 入力:** 傍点の後ろにカーソルを置いて IME で文字を入力すると、変換中の文字に傍点が付いて見えます。変換を確定すると傍点は消えます。
- **ルビ・縦中横の先頭・末尾での文字削除:** ルビ、縦中横に対して、その先頭・末尾にカーソルを置いて Delete キー・Backspace キーを押下しても文字を削除できません。文字列を選択してから削除するか、インライン展開した状態で削除してください。
- **青空記法の連続入力:** 青空記法に続けて別な青空記法を入力すると、後ろの青空記法が正しく挿入されなかったり、カーソル移動がおかしくなる場合があります。
- **行頭の青空記法への入力:** 行頭に青空記法がある場合、行頭にカーソルを置くと自動的にインライン展開されます。この状態で文字を入力すると記法の内側に挿入されますが、展開を閉じると入力した文字は通常のテキストとして扱われます。

## 設定

「設定 → TATE」から以下の項目を変更できます。

| 項目 | 説明 | デフォルト |
|------|------|-----------|
| フォントファミリー | 縦書き表示に使うフォント（CSS font-family 形式） | Hiragino Mincho ProN 系 |
| フォントサイズ | 縦書きビューのフォントサイズ（px） | 22 |
| 半角スペースを全角スペースに変換 | スペースキー入力を全角スペース `　` に変換 | ON |
| 行頭入力時に自動字下げ | 行頭で文字を入力したとき `　` を1文字自動挿入 | ON |
| 字下げを前の段落に揃える | 改行時、直前の段落と同じ数の `　` を自動挿入 | ON |
| 行頭カッコのインデントを削除 | スペースの後ろに全角開きカッコを入力したとき先頭の `　` を1文字削除 | ON |
| 禁則処理 | 行頭・行末の文字制限ルール | Normal |
